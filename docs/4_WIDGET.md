AI Agent Note — Premium Home Screen Widgets System
Goal

Design a set of extremely polished home screen widgets that match the app’s premium minimal identity.

The widgets should feel:

Elegant
Functional
Lightweight
Native-quality
Theme-aware
Visually refined

NOT:

Cheap Android widgets
Generic rectangular music controls
Cluttered utility panels

The widget system should feel like:

A natural extension of the app’s visual identity.

WIDGET SYSTEM STRUCTURE

Create:

One signature widget style for each theme
Each widget should have its own personality
But all should maintain consistent UX philosophy

Themes:

Monochrome Dark Widget
Monochrome Light Widget
Nothing OS Widget
GENERAL WIDGET PRINCIPLES
MUST FEEL PREMIUM

Widgets should:

Have balanced spacing
Smooth rounded corners
Proper typography hierarchy
Minimal information density
Clean interaction states

Avoid:

Tiny cramped controls
Too much metadata
Excessive borders
Loud gradients
WIDGET CONTENT
Required Elements
Essential
Album art
Song title
Artist
Play/pause
Next/previous
Playback progress
Optional
Tiny waveform
Subtle reactive ring
Playback animation
INTERACTION REQUIREMENTS
Widget interactions must feel instant

Actions:

Play
Pause
Skip

must:

Respond immediately
Animate smoothly
Never feel delayed
